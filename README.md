# ogorod-demo-vault

Синтетический демо-vault для публичного демо Огорода — [ogorod.ringova.com](https://ogorod.ringova.com).
Все проекты, задачи и дайджесты внутри вымышлены и собраны специально для показа
фич дашборда (канбан, RICE, роадмап, аналитика по грейдам, ручная грядка).
Никакого реального продукта или человека за этими данными нет.

Структура и канон документации — по шаблону [ogorodush](https://github.com/kringova/ogorodush)
(методология «Огород»). Смотри там `_templates/`, `docs/methodology.md` — этот vault
их не меняет, только заполняет реальным (синтетическим) контентом.
