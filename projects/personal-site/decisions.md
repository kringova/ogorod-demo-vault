## 2026-06-18 — Markdown-файлы вместо headless CMS

- **Контекст:** нужен способ редактировать посты без ручной правки HTML; кандидат — Sanity.
- **Решение:** остаёмся на markdown-файлах в репозитории, редактор — обычный текстовый/IDE.
- **Альтернативы:** headless CMS (Sanity) — отвергли: лишний сервис, отдельная авторизация
  и API-ключ ради UI, которым, кроме меня, никто не пользуется.
- **Следствия:** публикация поста — это `git push`, без визуального редактора; зато нет
  внешней зависимости и абонентской платы.
- **Статус:** принято

## 2026-05-08 — Astro вместо Next.js

- **Контекст:** сайт целиком статический (блог + портфолио), интерактивности почти нет.
- **Решение:** Astro — по умолчанию рендерит в чистый HTML, JS только там, где явно нужен.
- **Альтернативы:** Next.js — отвергли: тяжелее для чисто статического кейса, лишний
  клиентский JS на страницах, где ничего не интерактивно.
- **Следствия:** деплой проще и быстрее грузится; если понадобится динамика (форма,
  поиск) — придётся отдельно подключать island-компонент.
- **Статус:** принято
